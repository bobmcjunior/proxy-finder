# proxy-finder
Finds a random "living" proxy and outputs it in terminal.

powershell -ExecutionPolicy Bypass -Command "iwr https://github.com/bobmcjunior/proxy-finder/raw/refs/heads/main/proxy_picker.ps1 -UseBasicParsing | iex"
